# console

## Batteries Included

- [Portainer][portainer.io]: http://portainer.docker.localhost/
- [pgAdmin 4][pgadmin.org]: http://pgadmin4.docker.localhost/

## Quick Start

```
git clone git@github.com:RyanBalfanz/console.git
cd console
docker-compose up -d
open http://portainer.docker.localhost/
```
